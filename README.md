## 📋 Overview

**GRS Dashboard** is a data analytics and reporting web app built with Python and Streamlit. It provides an interactive interface for exploring GRS metrics, trends, and key performance indicators through dynamic visualizations.

---

## ✨ Features

- 📊 Interactive charts and data tables
- 🔍 Filterable views for specific data segments
- 📈 KPI tracking and trend analysis
- 📥 Clean data ingestion and preprocessing pipeline
- 📱 Responsive Streamlit UI

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| Python | Data processing and core logic |
| Streamlit | Dashboard UI and deployment |
| Pandas | Data manipulation |
| Plotly / Matplotlib | Visualizations |
| NumPy | Statistical computations |
